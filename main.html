Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML - Returns or sets HTML content inside an element
2. innerText - Returns or sets visible text inside an element
3. textContent - Returns or sets all text inside an element

Question-3: What is event delegation in the DOM?

Event Delegation = One parent event listener handles multiple child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model) where an event starts from the target element and propagates upwards through its ancestor elements.
